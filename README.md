# 2025 Merry Christmas

一個送給交換禮物夥伴的純前端小網頁，主題是「Make People Happy」。
使用 Vue 3 + TypeScript + Naive UI + UnoCSS，讓抽到藍芽喇叭的同事可以分享快樂、選擇喜歡的音樂氛圍。

## 啟動方式

1. 安裝套件

```bash
npm install
```

2. 開發伺服器

```bash
npm run dev
```

3. 建置產出

```bash
npm run build
```

## 功能亮點
- 聖誕風格的開場插畫，迎接交換禮物流程。
- 兩題關於「快樂」與「音樂喜好」的問卷，最後呈現恭喜領取藍芽喇叭的訊息。
- 使用 Pinia + VueUse 記錄答案並保留進度、Naive UI 與 UnoCSS 打造玻璃擬態聖誕介面。
